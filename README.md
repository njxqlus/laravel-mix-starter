# Laravel Mix Starter Project

## What's inside?

It's a pretty simple project, it's a starter kit based on [Laravel Mix](https://github.com/JeffreyWay/laravel-mix)

## Installation

1) `git clone https://github.com/njxqlus/laravel-mix-starter.git`

2) `npm install`
